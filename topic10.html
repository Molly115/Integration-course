CSS Div Tag

Mastering the ultimate generic container for web layouts and CSS art.

What is a Div?

Short for "division", the <div> tag groups sets of content together. It is a block-level element, meaning it starts on a new line and takes up the full width available.

  • Multi-purpose generic container.
  • Essential for CSS styling and JS manipulation.
  • The backbone of non-semantic layouts.

When to use it?

While semantic tags (header, footer, section) are preferred for meaning, the div tag is your primary tool for:

  • Web Layouts: Grouping sections like sidebars and main content.
  • CSS Art: Creating shapes and graphics using only code.
  • Spacing: Wrapping elements to apply padding or margins.

The Div Structure

<div class="container">
  <div class="header">Navigation Content</div>
  <div class="main">Article Text</div>
</div>

CSS Art: The Egyptian Flag

As detailed in your training file, you can create the Egyptian flag by nesting three div tags and styling them with specific colors.

.egypt-flag { width: 300px; height: 200px; }
.first-red { background-color: #ce1126; }
.white { background-color: #ffffff; }
.second-black { background-color: #000000; }
