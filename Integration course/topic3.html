CSS Basics

Mastering selectors, syntax, and the cascading nature of web styling.

What is CSS?

Cascading Style Sheets (CSS) is the language used to style an HTML document[cite: 526, 529]. It describes how HTML elements should be displayed on screen, paper, or in other media[cite: 528, 530].

  • Saves a lot of work by controlling multiple pages at once[cite: 530, 531].
  • Separates content (HTML) from presentation (CSS)[cite: 566].
  • Created by the W3C to solve formatting nightmares in early HTML[cite: 565].

CSS Syntax

A CSS rule consists of a selector and a declaration block[cite: 575].

h1 {
  color: blue;
  font-size: 12px;
}
  • Selector: Points to the HTML element you want to style[cite: 584].
  • Declaration: Contains a property name and a value[cite: 587].

Types of Selectors

CSS selectors are used to "find" the HTML elements you want to style[cite: 602]. They are divided into five main categories[cite: 603]:

  • Simple Selectors: Select based on name, id, or class[cite: 604].
  • Combinator Selectors: Select based on relationships between elements[cite: 605].
  • Pseudo-class Selectors: Select based on a certain state (e.g., hover)[cite: 606].
  • Pseudo-elements Selectors: Style a specific part of an element[cite: 607].
  • Attribute Selectors: Select based on an attribute or its value[cite: 608].

The ID Selector (#)

Uses the id attribute to select one unique element[cite: 686, 687].

#paral {
  text-align: center;
  color: red;
}

Note: An ID name cannot start with a number[cite: 700]!

The Class Selector (.)

Selects HTML elements with a specific class attribute[cite: 711].

.center {
  text-align: center;
  color: red;
}

Note: A class name cannot start with a number[cite: 717]!

Universal Selector (*)

The universal selector * selects all HTML elements on the page[cite: 732].

* {
  color: blue;
}

Grouping Selectors

Used to minimize code when multiple elements share the same style[cite: 766]. Separate selectors with a comma[cite: 768].

h1, h2, p {
  text-align: center;
  color: red;
}

Summary Checklist

  • CSS stands for Cascading Style Sheets[cite: 529].
  • Use # for IDs and . for classes[cite: 688, 712].
  • External stylesheets are stored in .css files[cite: 531].
  • Declaration blocks are surrounded by curly braces {}[cite: 588].
