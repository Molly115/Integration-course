Forms & Input

Building interactive user interfaces and styling form controls.

Essential Elements

Forms allow users to enter data which is then sent to a server for processing.

  • <form>: The container for all form elements.
  • <label>: Defines a caption for an item.
  • <input>: The most used element; can be text, radio, checkbox, etc.
  • <textarea>: For multi-line text input.

The <button> Element

Accepts a type attribute with three values:

  • submit: Sends form data to the page defined in the action.
  • reset: Returns all widgets to default values.
  • button: A generic button (used with JavaScript).

Organization: Fieldset & Legend

The <fieldset> tag is used to group related data in a form, and <legend> defines a caption for that group.

<fieldset>
  <legend>Employee Details</legend>
  First name: <input type="text">
</fieldset>

Form Styling (CSS)

Forms can be difficult to style. Use these common CSS properties to improve UX:

form { margin: 0 auto; width: 400px; }
input, textarea { font: 1em sans-serif; width: 100%; }
button { padding: 5px 10px; border-radius: 2px; }
