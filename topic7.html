Lesson 07

Images & Graphics

Working with standard images, Canvas drawing, and Scalable Vector Graphics.

The <img> Element

Used to embed a simple image file (JPG, PNG, GIF) into the webpage.

<img src="scream.jpg" alt="The Scream" width="220" height="277">
  • src: Path to the image file.
  • alt: Alternate text for accessibility.

HTML Canvas

The <canvas> element is a container used to draw graphics "on the fly" via JavaScript.

  • Used for paths, boxes, circles, and text.
  • Highly performant for games and real-time data visualization.
  • Requires a script to render content.

Scalable Vector Graphics (SVG)

SVG defines vector-based graphics in XML format. Unlike standard images, they do not lose quality when resized.

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="green" fill="yellow" />
</svg>

Canvas vs. SVG

[Image of difference between raster and vector graphics]
Feature Canvas SVG
Resolution Resolution dependent (can blur) Resolution independent (stays sharp)
Rendering Via Scripts (JavaScript) Via XML (Tags)
Interactivity Poor (no individual element events) Excellent (supports DOM events)
Performance Better for large numbers of objects Slower with too many objects
