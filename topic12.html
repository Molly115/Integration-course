Advanced Layout: Dropdowns

Creating interactive selection menus using standard and hoverable methods.

Standard Dropdown (Select)

The simplest way to create a dropdown is using the <select> and <option> elements.

  • Label: Provides context for the menu.
  • Select: The container for the menu.
  • Option: Individual items within the list.

User Experience

Dropdowns save space by hiding options until they are needed. They are essential for forms and navigation bars.

Step-by-Step Implementation

<label for="students">Choose a name:</label>
<select name="students" id="students">
  <option value="Lena">Lena</option>
  <option value="Mohamed">Mohamed</option>
</select>

Hoverable CSS Dropdowns

Your document also highlights a more advanced "hoverable" menu used in navigation. This uses position: absolute to show/hide content.

The CSS Logic

  • Set parent to position: relative.
  • Set content to display: none and position: absolute.
  • Use :hover to change display to block.

Example Style:

.dropdown:hover .content { display: block; }
